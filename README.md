# Flow - web-приложение магазина цветов

## Стек технологий:
JavaScript

**Frontend**: React, Redux, HTML, CSS

**Backend**: Node.js, Express, JWT, Bcrypt, Cors

**Database**: PostgreSQL, Sequelize ORM

### Установка:
* **server** - backend. 
   * cd server
   * npm i
   * npm run init
   * npm run dev
* **client** - frontend. 
   * cd client 
   * npm i
   * npm start
